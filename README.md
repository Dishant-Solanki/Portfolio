# AR Unity Project

## 📌 Project Overview

This project is an **Augmented Reality (AR) application built with Unity**.
The app allows users to place and interact with 3D objects in the real world using their mobile camera.

## 🎮 Features

* AR object placement on detected surfaces
* Interactive 3D models
* Real-time camera tracking
* Mobile AR experience

## 🛠 Technologies Used

* Unity Engine
* AR Foundation
* C#
* Android / iOS support

## 📱 How to Run the Project

1. Clone the repository
2. Open the project in **Unity Hub**
3. Use Unity version (example: 2022.3 LTS)
4. Connect an Android device
5. Build and run the project

## 📂 Project Structure

Assets – Game assets, scripts, and prefabs
Packages – Unity package dependencies
ProjectSettings – Unity project settings
